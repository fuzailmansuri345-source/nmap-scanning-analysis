# Advanced Nmap Scanning & Result Analysis

## 📌 Overview
This project focuses on performing advanced network scanning using Nmap to discover open ports, running services, and potential security vulnerabilities.

## 🎯 Objective
To understand how attackers and security professionals use network scanning to gather information about target systems.

## 🛠️ Tools Used
- Nmap
- Kali Linux

## 🔍 Techniques Used
- TCP Connect Scan
- SYN Scan
- UDP Scan
- Service Version Detection
- OS Detection
- Port Enumeration

## 📊 Analysis
The scan results were analyzed to identify:
- Open ports
- Running services
- Possible vulnerabilities
- Misconfigured services

## 📄 Report
A detailed analysis report of the scanning results is available in this repository.
